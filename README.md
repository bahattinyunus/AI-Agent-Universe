## 🚀 Proje Vizyonu

Bu repository, modern yapay zeka mühendisliğinin en uç noktası olan **Otonom Ajan Sistemleri** üzerine bir AR-GE laboratuvarıdır. Amaç sadece teorik bilgi değil, gerçek dünya problemlerini çözebilecek, hata toleransı yüksek ve ölçeklenebilir ajan mimarileri inşa etmektir.

Burada, "Static Code" devrinin bitip "Dynamic Cognition" devrinin başladığı noktadayız.

---

## 🏗️ Mimari Derinlik

Bir AI Agent'ı "Zeki" kılan yapı taşları bu repoda şu şekilde ele alınır:

1.  **🧠 Advanced Reasoning:**
    *   *Chain of Thought (CoT)* ile adım adım mantık yürütme.
    *   *Tree of Thoughts (ToT)* ile alternatif senaryo analizleri.
    *   *Reflexion* ile kendi hatasını fark edip düzelten loop'lar.

2.  **💾 Cognitive Memory:**
    *   **Short-term:** Bağlam içi öğrenme (In-context learning).
    *   **Long-term:** Vektör veritabanları (RAG) ile sonsuz hafıza kapasitesi.

3.  **🛠️ Skill & Tool Use:**
    *   API Entegrasyonları (Search, Database, Third-party Apps).
    *   Code Execution (Python Interpreter ile anlık kod çalıştırma).
